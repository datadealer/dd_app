# dd_app Documentation#

*ABANDONED*, see README.md for instructions

## Build ##

### Install Sphinx ###

Activate your virtual environment:

    $ source /path/to/venv/bin/activate

Install Sphinx

    $ pip install sphinx

### Build documentation ###

In `dd_app/doc`, run:

    $ make html

To build epub:

    $ make epub

## Access documentation ##

Point your browser to `dd_app/doc/html/index.html`
